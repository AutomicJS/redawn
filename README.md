# Redawn 😀

Redawn is a tiny, yet feature-packed Markdown alternative, designed for fast, in-browser rendering with more syntax options than Markdown, in under 10 KB unminified. 

## Files

1. dawn.js is the Object version of Redawn
2. dawn.es6.js is the ES6 version of Redawn 

## Syntax we support?

- Code
- Lists
- Bold
- Italics
- Links
- Images
- Page References (Anchor Tagging)
- Custom Elements
- & more

You can also:
  - Import and save files from GitHub, Dropbox, Google Drive and One Drive
  - Drag and drop markdown and HTML files into Dillinger
  - Export documents as Markdown, HTML and PDF
